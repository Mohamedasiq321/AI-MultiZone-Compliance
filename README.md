AI Multi-Zone Safety & Compliance Monitoring System

Overview

This project enables organizations to track safety adherence across different zones using computer vision. It generates instant alerts with structured data, helping teams respond quickly and maintain compliance standards.

Key Features

AI-Based Detection – Identifies safety violations in real time
Multi-Zone Monitoring – Separate tracking for each workspace zone
Live Alerts – JSON-based alerts with timestamp, zone ID, and confidence score
Real-Time Communication – WebSocket + REST integration
Role-Based Access – Admin and zone-specific monitoring
Interactive Dashboard – Live updates with a clean UI

Tech Stack

Backend: FastAPI, MongoDB
Frontend: React, Tailwind CSS
AI/Computer Vision: YOLO, OpenCV
Communication: WebSockets, REST APIs

How It Works

Video input is processed using AI models
Violations are detected per zone
Alerts are generated with structured data
Dashboard updates in real time for monitoring

Purpose

Built as a final-year project to demonstrate scalable, real-time safety monitoring using AI in industrial or organizational environments.
